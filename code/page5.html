MÉTIERS

Métiers du jeu vidéo, métiers d’avenir

Les opportunités de métiers dans le secteur du jeu vidéo sont multiples : tech, art, business. Souvent le jeu vidéo est à la pointe de la technologie et inspire d’autres secteurs (réalité virtuelle, expérience immersive, web3, …). Avec l’aide de +100 professionnels nous avons créé +80 fiches métiers du jeu vidéo. La lecture de ces fiches métiers est un bon début pour que votre enfant puisse se projeter dans son avenir et que vous puissiez l’y accompagner.
